Map-reduces operations:
Start your MongoDB shell by typing mongo in your terminal.

Select/Create the Database:
use dhanashri

Insert Sample Data (if not already inserted):

Define and Execute Map and Reduce Functions in One Go
db.employee.mapReduce(
    function() {
        emit(this.dept_name, this.salary);
    },
    function(keyDept, values) {
        return Array.sum(values);
    },
    { out: "total_salary_by_dept" }
)

View the Results: After executing the MapReduce command, you can check the results:
db.total_salary_by_dept.find().pretty()
